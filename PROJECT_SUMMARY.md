## 📦 What Users Will Need

After cloning your repo, users need to:
1. Clone ImGui: `git clone https://github.com/ocornut/imgui.git`
2. Build: `mkdir build && cd build && cmake .. && cmake --build . --config Release`
3. Run: `./Release/LimbusCompanyTrainer.exe`
